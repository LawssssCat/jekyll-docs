jemoji

jemoji is a GitHub-flavored Emoji plugin for Jekyll.

Configuration

Gemfile

gem "jemoji", "~> 0.8"

_config.yml

plugins:
  - jemoji

gfm_emojis

gemoji library contains character information about native emojis.

source code lib/kramdown/parser/gfm.rb in parser-gfm

details:

EXTENSIONS = {
  emoji: 'gfm/emoji_parser',
}.freeze

def initialize(source, options)
  ...
  return unless @options[:gfm_emojis]

  self.class.load_extension(:emoji) && @span_parsers.unshift(:emoji)
end

Configuration

Gemfile

gem "gemoji", "~> 3.0"

_config.yml

kramdown: 
  gfm_emojis: true
